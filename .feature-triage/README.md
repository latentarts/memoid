# Feature Triage

This folder is for feature triage notes and rough exploration that should not be treated as part of the Memoid engine, memory, or protocol system.

Guidelines:

- Keep notes here if they are for later consideration only.
- Do not reference this folder from `memory/`, `protocols/`, or `README.md`.
- `AGENTS.md` may define controlled workflows for reviewing or promoting notes from this folder.
- Nothing in this folder is part of the active Memoid workflow unless explicitly promoted later.
- Use `TEMPLATE.md` for notes that may eventually be promoted into GitHub issues.
