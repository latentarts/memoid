# Filing

Use this skill when the current session produced knowledge worth keeping.

## Read First

- `protocols/FILING.md`
- `protocols/FACTS.md` when durable facts changed

## Process

1. Update or create a session note under `evidence/sessions/`.
2. Update affected canonical wiki pages.
3. Update agent diaries when the lesson is specialist-specific.
4. Update fact pages when current reality changed.
5. Update `wiki/ESSENTIAL_STORY.md` only if startup context should change.

## Rules

- File only durable knowledge.
- Avoid duplicates and low-confidence speculation.
- Prefer updating existing canonical pages over spawning parallel versions.
