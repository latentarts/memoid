# Compaction

Use this skill before context is lost.

## Read First

- `protocols/COMPACTION.md`

## Process

1. Write or update a session note under `memory/evidence/sessions/`.
2. Record unresolved items.
3. Record decisions and rationale.
4. Note the pages that need follow-up edits.
5. Update the relevant agent diary if needed.

## Rules

- Prefer short, accurate handoffs.
- Capture work that would be expensive to reconstruct.
- If nothing durable happened, record that explicitly.
