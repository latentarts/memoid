# Retrieval

Use this skill for questions that should be answered from repository memory.

## Read First

- `protocols/RETRIEVAL.md`

## Retrieval Ladder

1. `memory/wiki/INDEX.md`
2. relevant wiki pages
3. linked evidence pages
4. raw sources

## Output

Answer using the smallest relevant set of pages and cite what was used.

## Rules

- Prefer canonical wiki pages over scattered mentions.
- Use evidence when chronology, proof, or precision matters.
- Use raw sources only when maintained layers are insufficient.
- If the answer creates durable synthesis, suggest or perform filing.
