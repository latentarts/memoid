# Retrieval Discipline

## Summary

The agent should not preload everything and should not answer from vague prior context when the repo can be checked directly.

## Key Ideas

- bounded default context reduces noise
- canonical pages should be consulted before evidence files
- evidence files should be consulted before raw sources
- durable answers should cite what was used

## Tradeoffs

- slower than bluffing from memory
- much more reliable than bluffing from memory
