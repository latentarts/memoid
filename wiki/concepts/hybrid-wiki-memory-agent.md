# Hybrid Wiki Memory Agent

## Summary

A markdown-first architecture that uses the Karpathy wiki pattern as the main knowledge artifact and MemPalace-style operational discipline to make it reliable for agent memory.

## Key Ideas

- wiki-first synthesis
- immutable raw sources
- bounded wake-up
- evidence-backed answers
- specialist diaries
- current-vs-history fact handling

## Variants

- pure Karpathy wiki: simpler, more synthesis-oriented
- pure MemPalace translation: more operationally strict, less lightweight
- hybrid model: wiki-centered with disciplined retrieval and filing

## Tradeoffs

- better reliability than a freeform wiki
- lower complexity than a database-first memory system
- weaker semantic search than an embedding-backed system
